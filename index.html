02.Projects that I've built.

Featured Project

BrewFinder

This website is designed to have a user search for a brewery in the city/town they are in or planning to go to. Once the website loads, a user can input the city/town in the text box and the API will load all of the breweries in the town or city. If they enter a city/town without any breweries, they will receive an error message asking them to try again.
