Featured Project

BrewFinder

This website is design to have a user search for a brewery in the city/town they are in or planning to go to. Once the website is load it will have the user put the city/town in the text box. Then they will hit find and the API will send all of the breweries listed in the town or city. If they have enter one that doesn't exist they will receive an error message asking them to try again.
